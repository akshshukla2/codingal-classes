About Me

A little bit about me

Hi, I’m Akshita, a passionate and curious individual driven by a deep love for learning and problem-solving. With a strong interest in technology and innovation, I’m constantly exploring ways to build, improve, and create through code. Whether it's developing small tools to automate tasks or diving into larger projects that challenge my thinking, I find immense satisfaction in turning ideas into working solutions.

My journey into coding began with curiosity, but quickly became a genuine passion. There's something incredibly rewarding about writing lines of code and watching them come to life—solving real-world problems or creating something entirely new. I enjoy the logical structure, the creativity involved, and the endless opportunities to grow. I especially love working on projects that push me to learn new languages, frameworks, or algorithms.

I’m excited for what’s ahead and always open to new challenges and opportunities to learn and grow.

My Projects :

Messaging App:

I created a simple message app that allows users to send and receive text messages in real-time. It uses Firebase for backend and authentication. The app features a clean UI, message history, and basic user management. It was my first step into real-time communication and cloud-based services.

Chat App:

I developed a real-time chat app where users can join rooms and communicate instantly. Built with React, Node.js, and Socket.io, it supports multiple users, message timestamps, and live updates. This project helped me understand WebSockets, event handling, and creating dynamic front-end experiences using modern JavaScript frameworks.

Portfolio App:

My portfolio app showcases my projects, skills, and background in a clean, responsive design. Built with HTML, CSS, and JavaScript (and React for the latest version), it highlights my coding journey and includes links to GitHub and live demos. It reflects my personality, passion, and commitment to web development.
